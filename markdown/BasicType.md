<h2 id='basicType'>基本語法</h2>  

> <h3 id='header'>標題</h3>
使用 # 字號當作標題大小, h1 ~ h6  
```  
# 標題1
## 標題2
```

---
> <h3 id='list'>清單</h3>
使用 - or 1-n or * 進行編號  

### 無序編號
- 清單1
    - 清單1-1
- 清單2
    - 清單2-1
### 有序編號
1. 清單  
    1. 清單1-1
1. 清單  
    1. 清單2-1

> <h3 id='hyperlink'>超連結</h3>

- 內部連結  
```
  [名稱](#連結)
```

- 外部連結  
```html
<a href="url“ target="_blank">hyperlink display</a>
```
